## Spinner Types

There are three types of spinners:
- **Default Spinner**: This is the default spinner that is used in the application, and uses `relative` positioning.
- **Absolute Spinner**: This spinner uses `absolute` positioning.
- **Fixed Spinner**: This spinner uses `fixed` positioning.
