## Getting Started

Getting started with **ARNIVA-CORE** is simple. The packages are designed to work independently, so you can install only what you need. To integrate it into your project, follow these steps:

First, install the core packages via npm:

```bash
npm install @arniverse/ui @arniverse/api @arniverse/utils @arniverse/tools
```

Once installed, you can import the components or utilities into your Svelte files:

```javascript
import { Button, Input } from '@arniverse/ui';
import { fetchData } from '@arniverse/api';
```
