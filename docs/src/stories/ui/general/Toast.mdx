## How to import
> ⚠️ Use lowercase `toast` when importing the component.

```js
import { toast } from '@ruzgardogu/ui';
```

## How to use
There are 6 types of toasts: `default`, `success`, `info`, `warning`, and `danger`.
> ⚠️ **IMPORTANT**: You need to call the `show` method to show the `default` toast.


```js
toast.show('This is a test message');
toast.success('This is a test message');
toast.info('This is a test message');
toast.warning('This is a test message');
toast.danger('This is a test message', 'custom-class');
```

### Setting Message
You can set the message of the toast by passing a string to the first parameter of the method. Or you can pass a configuration object to the second parameter.

```js
// Setting message as a string parameter
toast.show('This is a test message', config);
// Setting message as a configuration object
toast.show({
  message: 'This is a test message'
});
```

## Examples & Parameters
Toast component has multiple parameters:

```js
/** @type {ConfigProps} */
const defaultConfig = {
    type: 'info',
    timeout: 3000,
    position: 'top-right',
    showCloseButton: true,
    closeOnClick: true,
    pauseOnHover: true,
    showProgressBar: true,
    transition: 'slide',
    transitionDuration: 300,
    message: '',
    title: ''
}
```

### Parameters List
- `type`: Type of the toast. It can be `default`, `success`, `info`, `warning`, or `danger`.
- `timeout`: Time to close the toast in milliseconds.
- `position`: Position of the toast. It can be `top-right`, `top-left`, `bottom-right`, or `bottom-left`.
- `showCloseButton`: Show close button on the toast.
- `closeOnClick`: Close the toast when clicked.
- `pauseOnHover`: Pause the timeout when hovered.
- `showProgressBar`: Show progress bar on the toast.
- `transition`: Transition type of the toast. It can be `slide`, `fade`, or `bounce`.
- `transitionDuration`: Duration of the transition in milliseconds.
- `message`: Message of the toast.
- `title`: Title of the toast.
